# 🌍 **Vespera — Global Drinks & Social Spaces Explorer**  
*A UX-first, culturally aware database for understanding global drinking culture, safety expectations, and social environments.*

Vespera helps you explore:

- **What’s manufactured** — production, brands, and global availability  
- **What’s consumed** — cultural drinking patterns and norms  
- **What’s meaningful** — traditional beverages and social context  
- **Where it happens** — bars, events, and club types around the world  

This app is **not** designed to encourage drinking.  
It’s a guide to cultural norms, safety expectations, and social environments — whether you drink or not.

---

# ✨ **Features**
- Global drinks database with semantic filtering  
- Bar, event, and club types with consistent UX patterns  
- Safety and cultural context baked into every page  
- “Drank Everywhere” section for globally recognized brands  
- Mobile‑first, single‑column layout  
- Warm, semantic color system for emotional readability  
- Built with a design → engineering → refine workflow  

---

# 🛠️ **Tech Stack**
- **SvelteKit** (minimal template)  
- **TypeScript**  
- **Tailwind CSS**  
- **Semantic color system**  
- **Modular component architecture**

# 🚀 **Running the Project Locally**

Clone the repo:

```sh
git clone <your-repo-url>
cd project-root
```

Install dependencies:

```sh
npm install
```

Start the dev server:

```sh
npm run dev
```

Open the app in your browser:

```
http://localhost:5173
```

# 🧭 **Project Philosophy**
Vespera was built using a **UX-first workflow**:

- Major changes happen in design  
- Minor refinements happen in code  
- Figma Make was used early for exploration  
- The final UI was intentionally decluttered and rebuilt  
- Safety, culture, and context guide every design decision  

This project also unexpectedly strengthened my **DX skills**, helping me refine how I structure components, manage data, and build scalable patterns.
